# esg-performance-analysis
ESG vs Non-ESG Performance Analysis - Indian Equities (Jan 2022–Jan 2025) | Python · Plotly · NSE Data

Comparative study of 60 ESG vs 60 Non-ESG Nifty100 stocks across Jan 2022–Jan 2025.
Key findings:

Non-ESG outperformance driven by PSU capex supercycle (BEL, REC, PFC, Canara Bank) — not structural ESG underperformance
Return gap not statistically significant (independent samples t-test, p=0.198)
ESG stocks showed lower average volatility (24.9% vs 32.1%)

Tools: Python (pandas, scipy, yfinance, matplotlib) · Plotly · NSE / Yahoo Finance data
